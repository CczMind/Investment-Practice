## 1. 微分方程的基本定义

### 常微分方程 (ODE)
**核心定义**：只包含**一个自变量**的函数的导数的方程。
> A differential equation containing only one independent variable.

- **数学形式**：方程中只出现一个自变量（例如 `x` 或 `t`）的函数 `f` 及其导数。
  $$ F(x, f, f', f'', ...) = 0 $$
- **典型示例**：
  $$\frac{df}{dx} = -\frac{f(x)}{x}$$
  （这是一个ODE，因为函数 `f` 只依赖于一个变量 `x`）

### 偏微分方程 (PDE)
**核心定义**：包含**多个自变量**的函数的**偏导数**的方程。
> A differential equation containing multiple independent variables.

- **数学形式**：方程中出现多个自变量（例如 `t, x, y`）的函数 `u` 及其偏导数。
  $$ F(t, x, y, ..., u, u_t, u_x, u_y, u_{xx}, u_{tt}, ...) = 0 $$
- **典型示例**：
  Black-Scholes方程：$$V_t + \frac{1}{2}\sigma^2S^2V_{SS} + (r-q)SV_S - rV = 0$$
  （这是一个PDE，因为期权价格 `V` 是时间 `t` 和资产价格 `S` 两个自变量的函数）

**考试关键区分**：
> 看方程中的函数依赖于几个变量。一个变量 → ODE；多个变量 → PDE。

## 2. 定解条件：问题的"起点"与"边界"

### 初值问题
**核心定义**：在**时间维度**的**起始点**给出未知函数值的条件。

Forward Difference Scheme (Initial Value Problem)

- Step-by-step progression from a known point.
- Conditionally stable.
- Simple implementation.

### 边值问题  
**核心定义**：在**空间维度**的**边界**上给出未知函数值的条件。

Centered Difference Scheme (Boundary Value Problem)

- Requires two boundary conditions
- Transforms into a system of linear equations to solve
- Higher accuracy `(O(h^2))`

### 混合问题
**核心定义**：同时需要**初值条件**和**边值条件**。
- **典型示例**：
  Black-Scholes方程：
  - 初值条件（实际上是终端条件）：`V(T,S) = max(S-K, 0)`（到期日的支付）
  - 边值条件：`V(t,0) = 0`, `lim_{S→∞} V(t,S) = S - Ke^{-r(T-t)}`

- Terminal Condition:
$$
V(T,S) = (K-S)_+
$$
- Lower Boundary:
$$
V(t,0) = Ke^{-r(T-t)}
$$
- Upper Boundary:
$$
\lim_{S\to\infty}V(t,S) = 0
$$


**考试关键记忆**：
> 初值问题 → 时间维度，一个起点
> 边值问题 → 空间维度，多个边界
> 大多数金融PDE是混合问题

## 3. PDE的分类：基于数学特性的根本区分

这是PDE理论中**最重要**的分类方法，直接决定了应该使用何种数值方法。

### 分类标准：判别式 Δ = B² - 4AC

对于**二维二阶线性PDE**的一般形式：
$$ Au_{xx} + Bu_{xy} + Cu_{yy} + Du_x + Eu_y + Fu + G = 0 $$

通过计算判别式 `Δ = B² - 4AC` 来分类：

### 椭圆型方程 (Elliptic PDE)
- **判别式**：Δ < 0
- **物理意义**：描述**平衡态、稳态**系统，没有时间演化。
- **数学特性**：空间各向同性，信息向所有方向传播。
- **典型示例**：
  - Laplace方程：`u_xx + u_yy = 0`
  - Poisson方程：`u_xx + u_yy = f(x,y)`
- **金融应用**：某些稳态的资产定价问题。

### 抛物型方程 (Parabolic PDE)  
- **判别式**：Δ = 0
- **物理意义**：描述**扩散、耗散**过程，有时间演化但主要单向。
- **数学特性**：具有时间方向性，类似于热传导。
- **典型示例**：
  - 热传导方程：`u_t = αu_xx`
  - **Black-Scholes方程**：`V_t + (1/2)σ²S²V_SS + (r-q)SV_S - rV = 0`
- **金融应用**：**期权定价的核心方程**。

### 双曲型方程 (Hyperbolic PDE)
- **判别式**：Δ > 0  
- **物理意义**：描述**波动、传播**现象，信息沿特征线传播。
- **数学特性**：存在有限的传播速度。
- **典型示例**：
  - 波动方程：`u_tt = c²u_xx`
- **金融应用**：某些利率模型、波动率传播问题。

## 4. 金融中的具体应用与考试要点

### Black-Scholes方程的分类验证
对于Black-Scholes方程：
$$ V_t + \frac{1}{2}\sigma^2S^2V_{SS} + (r-q)SV_S - rV = 0 $$

计算判别式：
$$ Δ = B² - 4AC = 0² - 4 × [(1/2)σ²S²] × 0 = 0 $$

**结论**：Black-Scholes方程是**抛物型PDE**。

### 考试中必须掌握的对应关系

| PDE类型 | 判别式 | 物理过程 | 数值方法特点 | 金融示例 |
|---------|--------|----------|--------------|----------|
| 椭圆型 | Δ < 0 | 平衡态 | 需要求解大型线性系统 | 稳态定价 |
| **抛物型** | **Δ = 0** | **扩散过程** | **时间步进，稳定性关键** | **Black-Scholes** |
| 双曲型 | Δ > 0 | 波动传播 | 特征线法，迎风格式 | 波动传播 |

## 5. 线性与非线性PDE的区分

### 线性PDE
**定义**：未知函数 `u` 及其所有偏导数都以**线性形式**出现。
- **判断标准**：方程中不出现 `u`, `u_x`, `u_xx` 等的高次幂、乘积或其他非线性函数。
- **示例**：
  - `u_t + u_xx = 0`（线性）
  - `sin(xy)u_xx + 3x²u_xy + u_yy = 0`（线性，系数可以非常数）

### 非线性PDE  
**定义**：未知函数 `u` 或其偏导数以**非线性形式**出现。
- **判断标准**：包含 `u²`, `(u_x)³`, `u·u_xx`, `e^u` 等项。
- **示例**：
  - `u_t + u·u_x = νu_xx`（Burgers方程，非线性）
  - `u_xx + (u_y)² = f(x,y)`（非线性）

**金融意义**：大多数经典金融模型是线性的，但更复杂的模型（如随机波动率）可能产生非线性PDE。

## 考试答题模板

### 概念题回答示例：
"Black-Scholes方程是一个**二阶线性抛物型偏微分方程**。它是抛物型因为其判别式 `Δ = B² - 4AC = 0`；它是线性的因为未知函数 `V` 及其偏导数都以线性形式出现；它是一个混合问题，需要终端条件（在金融中代替初值条件）和空间边界条件。"

### 分类题回答步骤：
1. 识别方程中的最高阶偏导数项
2. 提取系数 `A, B, C`
3. 计算判别式 `Δ = B² - 4AC`
4. 根据Δ的符号确定类型
5. 检查线性性

这个部分的理解直接关系到后续数值方法的选择和稳定性分析，是考试中必须牢固掌握的基础。
